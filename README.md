# Personal Portfolio Website

This is my personal portfolio website built using HTML, CSS, and JavaScript.

The project is created to showcase my frontend skills, personal projects, and basic UI interactions while preparing for a Frontend Developer Internship.

## Project Overview

This portfolio includes:

-   Responsive navigation menu (mobile & desktop)
-   Scroll-based active link highlight
-   Blur header effect on scroll
-   Scroll-to-top button
-   Animated sections using AOS library
-   Responsive layout for multiple screen sizes
-   Contact form UI
-   Skills and experience sections

The project focuses on clean layout structure, reusable CSS classes, and smooth user interaction.

---

## Technologies Used

-   **HTML5**
-   **CSS3**
-   **JavaScript (Vanilla JS)**
-   **AOS (Animate On Scroll Library)**
-   **Remix Icons**
-   **Google Fonts**

---

## Key JavaScript Features

-   Show / hide mobile navigation menu
-   Remove mobile menu after clicking a link
-   Add blur effect to header on scroll
-   Highlight active navigation link based on section
-   Show scroll-up button when scrolling down
